#Sales_Data_Analysis_Python

# Overview
This project focuses on cleaning and preprocessing invoice data using Python and Pandas. The dataset was analyzed to identify and handle missing values, remove duplicates, correct data types, and prepare the data for further analysis or reporting.

# Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

# File Structure
- sales_analysis.ipynb: The Jupyter notebook containing the full data pipeline.online_retail.csv
- Raw transactional dataset.*.png
- Plots generated by the notebook (Top Products, Monthly Sales, Country Distribution, Top Customers, Weekly Distribution, and a Correlation Matrix).

# Data Cleaning Steps
- Imported the dataset
- Checked for missing values
- Removed or handled null values
- Converted date columns to datetime format
- Removed duplicate records
- Standardized column names
- Exported the cleaned dataset

# Feature Engineering & KPIs
- Calculates gross line-item revenue sales = Quantity * Price.
- Parses InvoiceDate to extract year, month, and weekday.
- Total Revenue: $10,642,128.80
- Average Item Value: $20.28
- Median Item Value: $11.90
